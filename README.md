## Bagian-Bagian Portfolio

### 1. Header & Navbar
- Navigasi tetap (fixed) di bagian atas
- Link ke semua bagian penting
- Logo/nama brand

### 2. Hero Section
- Foto profil dengan border lingkaran
- Judul dan deskripsi singkat
- CTA (Call to Action) button

### 3. About Section
- Informasi pribadi
- Quote/motivasi
- Deskripsi lebih lanjut

### 4. Projects Section
- Tampilan grid 3 kolom
- Kartu proyek dengan gambar
- Deskripsi dan link detail

### 5. Contact Section
- Form untuk pengunjung menghubungi
- Field: Nama, Email, Pesan
- Button submit

### 6. Footer
- Credit/attribution
- Link social media
- Copyright

## Customization Tips

**Ganti Foto Profil:**
- Ganti `rpi2.jpg` dengan foto Anda sendiri
- Letakkan file di folder yang sama dengan `index.html`

**Update Informasi:**
- Edit nama, sekolah, dan motivasi di bagian Hero dan About
- Tambahkan deskripsi proyek yang lebih detail

**Tambah Warna:**
- Ubah warna biru (`#0099ff`, `#0066cc`) di bagian `<style>` dengan warna favorit

**Social Media:**
- Update link Instagram di footer dengan akun Anda
